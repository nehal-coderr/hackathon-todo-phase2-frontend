# Hackathon Todo - Phase 2

A full-stack Todo application built for Hackathon Phase 2.  
This project includes a **Next.js frontend** and a **FastAPI backend**, allowing users to create, read, update, and delete tasks securely using JWT authentication.

---

## Features

- User authentication with JWT tokens
- Create, read, update, and delete tasks
- Responsive frontend built with Next.js
- Backend API built with FastAPI
- PostgreSQL database for task storage
- CORS enabled for secure frontend-backend communication
- Environment variables for secure configuration

---

## Tech Stack

- **Frontend:** Next.js, React, TypeScript
- **Backend:** FastAPI, Python 3.10+, asyncpg
- **Database:** PostgreSQL
- **Authentication:** JWT
- **Deployment:** Vercel (frontend), Render.com (backend)
- **Version Control:** Git & GitHub
